2. commit

git commit -m "comment" # easier

git reset HEAD~ -> rollback to stage / uncommit 

3. Log

git log --oneline -> it will show the changes in one line

5. Push to github

git branch -m main -> renaming the master into main

git push -u origin master / git push -u origin main -> push to github
